
PacketArt
=========
パケットアートのプログラムです。Javaで書かれています。

依存ライブラリ
=========
* Jnetpcap <http://jnetpcap.com/> (>=1.3)

動作条件(仮)
=========
* Java JRE (>=1.6)のインストール
* 1024x768以上の解像度のディスプレイ
* CPUi 1.6Ghz 2コア以上、メモリ2GB以上

インストール
=========
docディレクトリ参照。（未）

利用方法
=========
1. pcap(パケットダンプ)ファイルを<http://mawi.wide.ad.jp/mawi/>からダウンロードする
2. プログラムを走らせる
